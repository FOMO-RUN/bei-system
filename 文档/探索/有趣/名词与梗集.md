# 名词与梗集

用途：社区名词、绰号、梗的汇总。


## 中英对照（精选）

- Agentic Coding：智能体式编程  
- Skills：技能  
- Hooks：钩子  
- MCP（Model Context Protocol）：模型上下文协议  
- Vibe Coding：氛围式编程  
- TOON（The Object-Oriented Notation）：对象标记  
- Planner：规划者  
- Executor：执行者  
- Verifier：验证者  
- Prompt Cache：提示词缓存  

## 中英对照（扩展）

- Agent Runtime：智能体运行时  
- Tool Registry：工具注册表  
- Provider-agnostic：模型提供方无关  
- Multi-Agent：多智能体  
- Plan-Act-Observe：计划-行动-观察  
- ReAct：推理-行动循环  
- CoT（Chain of Thought）：思维链  
- ToT（Tree of Thought）：思维树  
- PDCA：计划-执行-检查-改进  
- DRY：不重复原则  
- KISS：保持简单原则  
- TDD：测试驱动开发  
- CI/CD：持续集成与交付  
- Evals：评估集  
- Worktree：工作树  
- Fractal Docs：分形文档  
- Gatekeeper：守门员  
- Context Window：上下文窗口  
- Deterministic Replay：确定性回放  
- Sandbox：沙盒
