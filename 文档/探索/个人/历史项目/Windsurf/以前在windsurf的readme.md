# 悖（BEI / Antinomy）

> 把中文目标变成可运行交付物，并生成可回放证据链与可复用技能

## 🎉 项目进展

### ✅ 已完成
- **全面中文化架构** - 采用workspace结构，所有包和目录使用中文命名
- **核心类型系统** - 完整的中文数据结构定义
- **CLI工具** - 功能完整的命令行界面
- **技能引擎** - 可扩展的技能系统，支持参数化执行
- **编排器** - 任务协调和执行管理
- **回放系统** - 执行记录和证据链管理
- **分形文档结构** - 完整的智能单文件协议实现，100%中文化文档

### 🚀 快速开始

```bash
# 构建项目
cargo build --release

# 查看帮助
cargo run --bin 悖 -- --help

# 列出可用技能
cargo run --bin 悖 -- 列出技能

# 执行中文目标
cargo run --bin 悖 -- 执行 --目标 "创建Hello World文件"

# 运行技能（需要JSON参数）
cargo run --bin 悖 -- 运行技能 --技能 创建文件 --参数 '{"文件名":"test.txt"}'
```

## 🔧 核心特性

### 中文主权优先
- 所有公共接口支持中文命名
- 完整的中文错误信息体系
- 中文宏封装：`打印!`, `格式化!`, `有!`, `成!`, `败!`
- 100%中文化的文档结构和文件命名

### 证据驱动开发
- 每次执行生成完整证据链
- 支持精确回放和验证
- 决策过程全程记录

### 技能系统
- 可扩展的技能注册机制
- 参数化技能执行
- 技能版本管理

### 分形文档架构
- 智能单文件协议实现
- 入口永远一处，上下文永远连续
- 禁止孤儿文件，按需读取
