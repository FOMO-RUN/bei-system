# 语言裁决账本

## 裁决记录

### 0001: 项目初始化目录命名
- **日期**: 2026-01-29
- **命名点**: 项目根目录结构
- **候选方案**: 
  1. 全中文目录（`ai地图/`、`模块/`、`文档/`）
  2. ASCII 目录 + 中文映射（`ai-map/`、`modules/`、`docs/`）
- **证据**: Windows 文件系统支持 Unicode，但部分工具可能存在兼容性问题
- **裁决结论**: 采用 ASCII 目录 + 中文映射（方案2）
- **策略**: 建立 `词汇映射表.yaml`，对人输出使用中文显示名
- **影响面**: 整个项目目录结构
- **回滚点**: 重新创建目录结构，更新映射表

### 0002: Git 仓库命名
- **日期**: 2026-01-29
- **命名点**: Git 分支、标签、提交信息
- **候选方案**:
  1. 全中文命名
  2. ASCII 命名 + 中文描述
- **证据**: Git 2.52.0 对 Unicode 支持良好，但部分 Git GUI 可能存在显示问题
- **裁决结论**: 采用 ASCII 命名 + 中文描述（方案2）
- **策略**: 分支名使用 ASCII，提交信息使用中文
- **影响面**: 版本控制流程
- **回滚点**: 重命名分支，修改提交信息模板

4. **可回滚性** - 每个决策都要有明确的回滚方案
