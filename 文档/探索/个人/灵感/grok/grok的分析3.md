2. 收集证据 (Rust docs / crates.io 规则)：技术支持 Unicode 吗？
3. 三级分类：
   - 🟥 不可失败 (错即崩)：保留 ASCII + 中文解释。
   - 🟨 高代价 (错代价大)：优先裁决，允许中文但需回滚方案。
   - 🟩 可即兴 (错可改)：必须中文，实时生成。
4. 不遗漏：用 walkdir + syn 递归扫描所有点，遗漏 = 升级 HCR。
5. 省 token：cache 裁决结果 (hashmap 存 “函数名:中文” )。

## 工作流程 (multi-agent 省 token)

3. 等待批准。

5.  跑测试/ e vals，提供证据 (subagent: verifier)。
